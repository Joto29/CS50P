# CS50P – Python Practice

This repository contains Python programs created while completing Harvard's CS50P course.

## Files
- `hello.py` – Prints a greeting to the user
- `calculator.py` – Performs basic arithmetic operations
- `greeting.py` – Greets the user based on input
- `guess.py` – Number guessing game
- `machine.py` – Simulates a simple vending machine
- `shows.py` – Demonstrates data processing and output

## How to run
Make sure Python 3 is installed, then run:

```bash
python hello.py
